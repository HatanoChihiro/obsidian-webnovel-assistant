# WebNovel Assistant

专为网文小说创作设计的 Obsidian 插件。

[![GitHub release](https://img.shields.io/github/v/release/HatanoChihiro/obsidian-webnovel-assistant)](https://github.com/HatanoChihiro/obsidian-webnovel-assistant/releases)
[![License](https://img.shields.io/github/license/HatanoChihiro/obsidian-webnovel-assistant)](LICENSE)

---

## 功能一览

### 字数统计
- 实时统计当前文档字数，适配中文创作平台的计算规则
- 文件树中显示文件夹和文档的汇总字数
- 高性能缓存，文件夹字数查询性能提升 95%+

### 写作目标
- 全局默认章节目标字数，支持单文件通过 frontmatter `word-goal` 单独设定
- 当日目标字数，统计今日所有文件的新增字数总和
- 状态栏实时显示完成进度

### 专注计时
- 自动区分专注打字时间与摸鱼时间，支持暂停/恢复
- 可配置闲置超时阈值
- Worker 线程处理，不阻塞主线程

### 悬浮便签
- 6 种预设配色主题，支持自定义背景色和文字色
- 从选中文字或文件直接创建便签
- 可关联文件、固定、缩放，状态持久化重启后恢复
- 保存时可自定义文件名和保存位置，关闭时智能提示保存

### 伏笔标注
- 选中文字后右键「标注为伏笔」或使用快捷键 Ctrl/Cmd+Shift+F
- 自动保存到当前文件夹下的伏笔文件（文件名可自定义）
- 支持多标签分类，相同说明的伏笔自动合并追加
- 侧边栏伏笔面板：按状态分组（未回收/已回收/已废弃），直接标记回收或废弃
- 回收时指定回收章节，形成完整的伏笔生命周期追踪

### 智能章节排序
- 自动识别章节编号并按数字大小排序
- 支持中文数字（第一章）、阿拉伯数字（第01章）、英文（Chapter 1）及卷/部/册/篇等单位
- 章节文件作为整体块排列，非章节文件（大纲、伏笔等）保持手动排序位置，兼容 manual-sorting 插件
- 可在设置中启用/禁用

### 合并章节
- 右键文件夹 → 合并章节
- 自动收集文件夹内的章节文件，排除大纲、伏笔等非章节文件
- 按章节编号顺序合并，显示总字数

### 自动生成下一章
- 基于当前文件名自动递增章节编号
- 支持中文数字和阿拉伯数字，不复制原章节标题

### 写作状态面板
- 侧边栏面板，显示今日目标进度、专注/摸鱼时长、近 7 日字数图表
- 点击图表可查看历史统计详情（日/周/月/年）

### OBS 直播叠加层
- 内置 HTTP Server，零延迟、零磁盘消耗
- 显示专注时长、摸鱼时长、章节目标进度、当日目标进度、本场净增
- 各模块可独立开关，支持自定义透明度和 CSS
- 兼容旧版 TXT 文件导出模式

### 护眼模式
- 将编辑区和阅读区背景替换为护眼色，其他界面保持不变
- 颜色可自定义，默认豆沙绿 #E8F5E9

### 移动端适配
- 手机端自动进入 Lite 模式，保留字数统计和目标设定
- 触摸优化，44px 最小触摸目标

## 安装

1. 从 [Releases](https://github.com/HatanoChihiro/obsidian-webnovel-assistant/releases) 下载最新版本的 `main.js` 和 `manifest.json`
2. 在 Obsidian 库中创建文件夹：`.obsidian/plugins/web-novel-assistant/`
3. 将下载的文件放入该文件夹
4. 重启 Obsidian，在设置 → 第三方插件中启用

## 使用指南

### 快捷命令

| 命令 | 说明 |
| :--- | :--- |
| 打开/关闭写作实时状态面板 | 切换右侧边栏状态面板 |
| 打开/关闭伏笔面板 | 切换侧边栏伏笔视图 |
| 开始/暂停码字与时长统计 | 切换计时状态 |
| 新建空白悬浮便签 | 创建空白便签 |
| 标注为伏笔 | 将选中文字标注为伏笔（Ctrl/Cmd+Shift+F） |
| 自动生成下一章 | 基于当前文件名递增编号 |
| 重置直播统计数据 | 清零时长与今日字数 |

### 右键菜单

- **文件**：设定目标字数 / 作为便签打开
- **文件夹**：合并章节
- **编辑器（有选中文字）**：标注为伏笔 / 将选中内容抽出为便签
- **编辑器**：设定目标字数

### Frontmatter 配置

```yaml
---
word-goal: 5000
---
```

### OBS 叠加层设置

1. 在插件设置中启用「启用 OBS 直播叠加层」
2. 在 OBS 中添加「浏览器源」，URL 填入 `http://127.0.0.1:24816/`
3. 建议尺寸：宽度 300px，高度 400px

## 设置项

| 设置 | 说明 | 默认值 |
| :--- | :--- | :--- |
| 显示章节目标进度 | 状态栏显示字数完成进度 | 开启 |
| 显示文件列表字数 | 文件树中显示字数 | 开启 |
| 默认章节目标字数 | 每章目标字数 | 3000 |
| 当日目标字数 | 今日新增字数目标 | 5000 |
| 启用智能章节排序 | 按章节编号自动排序 | 关闭 |
| 护眼模式 | 编辑区背景替换为护眼色 | 关闭 |
| 护眼背景色 | 自定义护眼颜色 | #E8F5E9 |
| 伏笔文件名 | 伏笔文件的名称 | 伏笔 |
| 闲置超时阈值 | 超过此时间无操作判定为摸鱼 | 60 秒 |
| OBS 服务器端口 | HTTP 服务器监听端口 | 24816 |
